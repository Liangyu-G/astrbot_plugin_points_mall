# AstrBot 积分商场插件 (astrbot_plugin_points_mall)

基于 AstrBot 框架开发的积分商城系统，支持群聊活跃度积分、签到积分，以及完善的商品管理流程。

## 核心功能

- **帮助菜单**：
  - `积分帮助`：查看积分商城全部指令。
- **框架 T2I**：
  - 支持通过 `use_t2i` 配置接入 AstrBot 文本转图片，长文本回复会按框架 T2I 规则转图。
- **积分获取**：
  - `签到`：每日随机获得积分。
  - **活跃奖励**：在群聊中发送超过指定长度（默认10字）的消息即可获得积分（带冷却防止刷屏）。
- **商场管理**：
  - `积分商场`：查看当前在售商品序号及价格。
  - `上架 [名称] [价格] [描述]`：管理员可申请上架，超级管理员直接上架。
  - `授权上架 [序号]`：超级管理员审核并发布商品。
- **个人中心**：
  - `我的积分`：查询当前余额。
  - `购买 [序号]`：**（仅限私聊）** 扣除积分、生成待发货订单，并私聊通知超级管理员和商品上架者。
- **管理权限**：
  - `给予积分 [QQ] [数额]`：超级管理员手动调整用户积分。
  - `订单列表 [待处理/已完成/已取消/全部]`：管理员查看最近订单。
  - `完成订单 [订单号]`：超级管理员或商品上架者标记订单完成，并通知购买人。
  - `取消订单 [订单号]`：超级管理员或商品上架者取消待处理订单并自动退款。

## 安装说明

1. 将本仓库克隆至 `data/plugins/` 目录。
2. 在 `_conf_schema.json` 或 AstrBot 管理面板中配置 `super_admins`（超级管理员）及 `admins`（普通管理员）的 QQ 号。
3. 重启 AstrBot 即可生效。

## 配置项说明

- `super_admins`: 超级管理员列表。
- `admins`: 普通管理员列表。
- `sign_min` / `sign_max`: 签到积分波动区间。
- `active_text_len`: 活跃文字字数阈值，消息文字数必须严格超过该值才奖励积分。
- `active_point_reward`: 每次活跃奖励的积分。
- `active_cooldown`: 两次活跃积分获取的间隔秒数。
- `use_t2i`: 是否让插件回复接入 AstrBot 框架 T2I，默认开启。
